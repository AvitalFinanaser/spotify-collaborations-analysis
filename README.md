# spotify-collaborations-analysis
 Analysis of artist collaborations in Spotify hits from 2000-2023.

## Overview

This project analyzes collaboration networks and lyrics in top Spotify songs between 2000-2023 using:
- Network analysis (SNA) to understand artist collaboration patterns
- Natural language processing (NLP) to analyze song lyrics

**Playlist url** = https://open.spotify.com/playlist/6Q1m6h43xSG5yiT4gzDHoW?si=4dd439e990974000

## Data Sources

Data was collected via Spotify API for playlist, songs and artists features.
In addition, Genius API was used for lyrics extraction.

## Key Findings

- Most popular artists have more collaborations
- Genre influences collaboration patterns
- Most songs (69.7%) express positive sentiment
- Lyrics frequently focus on love and relationships

## Future Work
